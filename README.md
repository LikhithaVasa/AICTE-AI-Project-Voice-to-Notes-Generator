# AI-Powered Lecture Transcription and Note Generation System

## 📌 Project Overview
This project automatically extracts audio from YouTube lecture videos, 
converts speech to text using OpenAI Whisper, 
and generates summarized notes for students.

## 🚀 Technologies Used
- Python
- OpenAI Whisper
- Google Colab
- NLP Techniques

## 🔧 Features
- YouTube video transcription
- Automatic text summarization
- Study note generation
- Multiple lecture support

## ▶ How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Provide YouTube link
4. Run all cells

## 📚 Future Scope
- Automatic PPT generation
- Quiz generation
- Web application deployment
